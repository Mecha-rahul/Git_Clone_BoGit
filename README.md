# Bogit

A simple Git clone built in Python to understand how version control systems work under the hood.

## About

Bogit is a personal learning project where I am recreating some of the core concepts of Git using Python. The goal is not to replace Git, but to gain a deeper understanding of how repositories, commits, object storage, and version tracking work internally.

While using Git in my projects, I became curious about what actually happens behind commands such as:

* `git init`
* `git add`
* `git commit`
* `git status`

To explore these concepts, I started building **Bogit**, a lightweight Git clone that helps me learn how version control systems work from the ground up.

## Tech Stack

* Python 3

## Disclaimer

Bogit is a learning project and is not intended for production use. It is built purely for educational purposes to better understand the design and implementation of Git.
